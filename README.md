# Final Year Project (FYP) Management System

A comprehensive web-based platform designed to streamline, automate, and centralize the Final Year Project (FYP) lifecycle for university students, faculty members, and administration.

## Aims & Objectives
The primary aim of this project is to bridge the administrative and communication gaps between students, project supervisors, committee members, and department coordinators. By digitalizing the entire FYP workflow—from group formation to final grading—the system reduces paperwork, ensures transparent evaluations, and helps keep project timelines on track.

## Current Features
The system implements a robust multi-role architecture tailored to the specific needs of all FYP stakeholders:

- **Student Portal:**
  - Dynamic group formation and management.
  - Project proposal submission and tracking.
  - Ongoing task management and continuous progress tracking.
  
- **Supervisor Portal:**
  - Proposal review workflow featuring inline, native PDF preview capabilities.
  - Granular, per-student "Supervision Marks" evaluation system.
  - Seamless tracking of assigned groups' progress.

- **Committee Member Portal:**
  - Comprehensive, individual per-student grading architecture for Proposal, Progress, and Final presentations.
  - Automated score validation, calculations, and mobile-responsive evaluation interfaces.

- **Department Coordinator Portal:**
  - Centralized dashboard for department-wide statistics.
  - Student registration and verification management.
  - System-wide notice and announcement broadcasting.
  - Dynamic **External Assessment Sheet** generation (CSV export with customizable attributes and shift filtering).

- **Real-Time Communication & AI:**
  - **Firebase Real-Time Chat:** Seamless, instant messaging between project groups and their supervisors. Features a highly polished, responsive WhatsApp-style UI with drag-and-drop document/image sharing, message truncation, and edit/delete/copy actions.
  - **AI Student Guidance:** An integrated Chatbot designed to assist students with FYP rules, deadlines, and technical queries.

- **System-Wide:**
  - Secure authentication and role-based access control.
  - Highly responsive, mobile-first modern user interface (Light & Dark modes).
